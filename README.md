# BadBankProject
## Banking application 

## Description
BadBank is a complete web app where I used a client, server, and database.
On the other hand, I used a Docker container for hosting my app.

## Technologies used
I used:
Full Stack MERN, Javascript, HTML, Docker, Bootstrap, Robo T3.
